Customisation du visage

1) Création du visage

Vous avez la possibilité de créer le visage de votre avatar lors de la création de votre compte sur l'application mobile. Pour cela, il vous faut cliquer sur l’icône se trouvant en haut à gauche de votre écran sur la page d'accueil :
► Afficher le texte

Puis il vous suffit de sélectionner les différents traits du visage ainsi que leurs couleurs.


2) Modification du visage

Une fois que vous avez créé le visage de votre avatar sur l'application mobile, il ne vous est plus possible de le modifier via l'application. Vous avez cependant la possibilité de changer son humeur à volonté. Pour cela vous devez cliquer sur votre avatar depuis la page d'accueil et sélectionner l’icône en haut à droite dans le menu qui apparaît :
► Afficher le texte


Customisation des équipements

1) Obtention des équipements

En jouant sur l'application mobile, vous pouvez débloquer des équipements pour votre avatar. Il existe plusieurs moyens d'en obtenir :
En les achetant dans la Boutique de l'application contre des cristaux : rendez-vous dans la section "Le Tailleur de Thiercelieux" depuis la Boutique.
En augmentant le niveau de votre profil, vous pouvez obtenir des équipements d'avatar en récompense pour votre progression.
En composant un code cadeau : pour cela rendez-vous dans les paramètres (icône en haut à gauche de la page d'accueil) et cliquez sur "Code cadeau" :
► Afficher le texte

Des équipements exclusifs peuvent être disponibles dans la Boutique à l'occasion de certains événements, soyez à l'affût pour ne pas les manquer !


2) Modification des équipements

Une fois vos équipements obtenus, il ne vous reste plus qu'à les attribuer à votre avatar. Il vous suffit de cliquer sur votre avatar depuis la page d'accueil de l'application mobile et de faire votre choix !

Attention : Si vous équipez une panoplie, certains équipements peuvent ne pas être compatibles.
